## 🚀 About The Project

This is a **premium personal portfolio** designed to showcase high-level expertise in Software Engineering, specifically tailored for Game Server Development (SA-MP & FiveM) and AI Integration.

Built with a focus on **performance**, **aesthetics**, and **user experience**, this portfolio features a stunning dark mode design with neon accents, glassmorphism effects, and smooth animations.

### ✨ Key Features

- **🎨 Premium Dark UI:** A sophisticated dark theme with vibrant neon accents (Cyan, Purple, Blue).
- **🪞 Glassmorphism Aesthetic:** Modern frosted glass effects with subtle grain noise textures.
- **🌍 Internationalization (i18n):** Full multi-language support:
  - 🇺🇸 English
  - 🇮🇩 Indonesian
  - 🇨🇳 Chinese
  - 🇸🇦 Arabic (with automatic RTL layout)
- **🔄 GitHub API Integration:** Dynamically fetches and displays the latest 6 repositories.
- **📱 Fully Responsive:** Flawless experience on Mobile, Tablet, and Desktop.
- **✨ Smooth Animations:** Powered by Framer Motion for elegant scroll reveals.
- **🎮 Specialized Content:** Dedicated sections for Game Server Development (Pawn/Lua) and AI.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- **Node.js** (v18 or higher)
- **npm** or **yarn**
- **Git**

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/jackdogle/jackdogle.github.io.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd jackdogle.github.io
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

5. **Open your browser**
   Visit `http://localhost:3000` to see the application.

## 📦 Deployment

This project is pre-configured for **GitHub Pages** deployment.

1. **Build the project**
   ```bash
   npm run build
   ```

2. **Deploy to GitHub Pages**
   ```bash
   npm run deploy
   ```

3. **Configure GitHub Settings**
   - Go to **Settings > Pages** in your repository.
   - Set **Source** to `Deploy from a branch`.
   - Select `gh-pages` branch and `/ (root)` folder.
   - Click **Save**.
